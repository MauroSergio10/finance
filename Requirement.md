RF1: O sistema deve autenticar o usuário utilizando e-mail e senha válidos
RF2: O sistema deve gerar um relatório mensal em PDF contendo o resumo financeiro do usuário.
RF3: O sistema deve permitir criar, editar e deletar cartões de crédito dentro de suas contas bancárias
RF4: O sistema deve notificar o usuário por e-mail quando uma transação exceder o valor determinado pelo mesmo.
RF5: O sistema deve notificar o usuário por e-mail quando o cartão estiver faltando 1 semana e quando estiver expirado.
RF6: O sistema deve permitir o usuário criar transações fulturas a data atual.
